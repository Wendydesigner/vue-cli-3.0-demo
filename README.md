# mpa-demo

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Start up assigned page
```
yarn run serve -e <file_name>

```
### Add page
```
yarn run patch -a <file_name>
```
### delete page
```
yarn run patch -d <file_name>
```
